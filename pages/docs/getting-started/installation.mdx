## Installation

TanStack Router는 [NPM](https://npmjs.com) 패키지 매니저를 사용하여 설치할 수 있습니다.

```sh
npm install @tanstack/react-router
# 또는
pnpm add @tanstack/react-router
# 또는
yarn add @tanstack/react-router
# 또는
bun add @tanstack/react-router
```

TanStack Router는 현재 React와 ReactDOM에서만 호환됩니다. React Native 어댑터에 기여하고 싶다면 [Discord](https://tlinz.com/discord)를 통해 문의해주세요.

### Requirements

- React v18.x.x
- ReactDOM v18.x.x
- TypeScript >= v5.2.x (TypeScript는 선택 사항이지만 권장됩니다)
  - TypeScript의 최근 5개의 마이너 버전을 지원하는 것을 목표로 합니다. 오래된 버전을 사용 중이라면 문제를 겪을 수 있습니다. 호환성을 위해 최신 버전의 TypeScript로 업그레이드해주세요.
  - 위에서 언급한 범위를 벗어난 오래된 TypeScript 버전에 대한 지원은 사소한 업데이트나 패치 릴리스에서 사전 경고 없이 중단될 수 있습니다.
