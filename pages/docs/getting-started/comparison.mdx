기능/지원 수준 키:

- ✅ 1급 지원, 내장되어 추가 설정이나 코드 없이 사용 가능
- 🔵 애드온 패키지를 통해 지원
- 🟡 부분 지원
- 🔶 가능하지만 사용자 정의 코드/구현/캐스팅이 필요
- 🛑 공식적으로 지원되지 않음
