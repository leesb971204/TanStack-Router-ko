## createRoute function

`createRoute` 함수는 [`Route`](./RouteType.md) 인스턴스를 반환합니다. 반환된 라우트 인스턴스는 루트 라우트의 자식으로 전달되어 라우트 트리를 생성하며, 이 트리는 라우터에 전달됩니다.

## createRoute options

- **타입**: [`RouteOptions`](./RouteOptionsType.md)
- **필수 항목**
- 라우트 인스턴스를 구성하는 데 사용되는 옵션입니다.

## createRoute returns

새로운 [`Route`](./RouteType.md) 인스턴스.

## Examples

```tsx
import { createRoute } from "@tanstack/react-router";
import { rootRoute } from "./__root";

const Route = createRoute({
  getParentRoute: () => rootRoute,
  path: "/",
  loader: () => {
    return "Hello World";
  },
  component: IndexComponent,
});

function IndexComponent() {
  const data = Route.useLoaderData();
  return <div>{data}</div>;
}
```
