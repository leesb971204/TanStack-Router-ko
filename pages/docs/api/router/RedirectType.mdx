## Redirect type

`Redirect` 타입은 TanStack Router에서 리디렉션 동작을 나타내는 데 사용됩니다.

```tsx
export type Redirect = {
  code?: number;
  throw?: any;
} & NavigateOptions;
```

- [`NavigateOptions`](./NavigateOptionsType.md)

## Redirect properties

`Redirect` 객체는 다음 속성을 포함하거나 허용합니다:

### `code` property

- 타입: `number`
- 선택 사항
- 리디렉션할 때 사용할 HTTP 상태 코드.

### `throw` property

- 타입: `any`
- 선택 사항
- 제공된 경우, 리디렉션 객체를 반환하는 대신 던집니다. 이는 함수에서 던지는 동작이 반환 타입을 `never`로 만들 수 있는 경우에 유용합니다. 이 경우 `redirect({ throw: true })`를 사용하여 객체를 반환하는 대신 던질 수 있습니다.
