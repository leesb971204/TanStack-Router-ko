## Navigate component

`Navigate` 컴포넌트는 렌더링 시 새로운 위치로 이동하기 위해 사용할 수 있는 컴포넌트입니다. 경로 이름, 검색 매개변수, 해시, 위치 상태 등의 변경을 지원합니다. 렌더링이 성공적으로 완료되면, 내부적으로 `useEffect` 훅을 사용해 네비게이션이 실행됩니다.

## Navigate props

`Navigate` 컴포넌트는 다음 속성을 받습니다:

### `...options`

- **타입**: [`NavigateOptions`](./NavigateOptionsType.md)

## Navigate returns

- `null`
