## ErrorComponent component

`ErrorComponent` 컴포넌트는 오류 메시지를 렌더링하며 선택적으로 오류의 메시지를 표시하는 컴포넌트입니다.

## ErrorComponent props

`ErrorComponent` 컴포넌트는 다음 속성을 받습니다:

### `props.error` prop

- **타입**: `any`
- 자식 컴포넌트에서 발생한 오류.

### `props.reset` prop

- **타입**: `() => void`
- 오류 상태를 프로그래밍 방식으로 재설정하는 함수.

## ErrorComponent returns

- 오류 메시지가 존재하면 형식화된 오류 메시지를 반환합니다.
- "Show Error" 버튼을 클릭하여 오류 메시지를 토글할 수 있습니다.
- 기본적으로, 오류 메시지는 개발 환경에서 표시됩니다.
